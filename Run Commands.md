# Run Commands

## Create venv
```
python -m venv venv
```

## Activate venv
```
.\venv\Scripts\Activate.ps1
```

## Setup Library
```
pip install -r requirements.txt
```

# Install <TNH_Optima>.spec
```
pyinstaller --clean --noconfirm TNH_Optima.spec
```
